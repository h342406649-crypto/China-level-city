# China-level-city
中国版图制霸：地级区划版

一个基于 Web 的中国地级行政区划旅行记录系统，让您可以标记和追踪在中国各地的旅行足迹。

功能特色

• 🗺️ 交互式地图：使用 ECharts 绘制精美的中国地图

• 📊 多级评分系统：对每个地区进行 0-4 级的详细评分

• 🌏 两级视图：全国视图和省级详细视图

• 📱 响应式设计：适配各种设备屏幕尺寸

• 💾 本地数据存储：所有数据保存在浏览器本地

• 📸 图片导出：支持将地图导出为高清图片

使用说明

等级定义

• 0级（没去过）：白色区域，表示从未到访

• 1级（途经/停留）：浅蓝色，表示短暂停留或途经

• 2级（玩过1-3天）：绿色，表示短期旅行（1-3天）

• 3级（旅居3-30天）：金色，表示较长时间居住（3-30天）

• 4级（居住30天以上）：红色，表示长期居住（30天以上）

操作方法

1. 全国视图：点击任意省份可查看该省详细地图
2. 省级视图：点击省内城市可设置旅行等级
3. 等级选择：点击地区后弹出等级菜单，选择相应等级
4. 返回全国：点击"返回全国"按钮可回到全国视图
5. 保存图片：点击"保存图片"可导出当前地图

技术栈

• 前端框架：原生 HTML5 + CSS3 + JavaScript

• 地图库：ECharts 5.4.3

• 地图数据：基于阿里云 Datav 的地理数据

• 部署平台：GitHub Pages
